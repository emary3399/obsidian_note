### 文本编辑提示词

#### 1.文本润色
```
请润色以下内容的语言表达，使其[效果]
```

#### 2. 文本扩写

```
请将以下内容扩写成一篇详细的文章，使其更具深度和丰富性
```

#### 3. 观点提炼

```
请从以下内容中提炼出核心观点，使其更加简洁明了
```

#### 4. 诗意描述

```
请用充满诗意的语言描述[对象]，赋予其独特的美感和情感
```

#### 5. 文本校对

```
请检查以下内容的语法、拼写和标点，指出错误并提供修改建议
```
